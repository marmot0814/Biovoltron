Installation {#installation}
============
In this short guide you will learn how to set up Biovoltron
and how to compile a small example to test whether
everything works.

Dependencies
------------
* g++ 11.1.0+
* boost 1.74.0+
* zlib 1.2.11+
* tbb 2020.3+
* range-v3 0.11.0+
* cmake 3.16+ (optional)
